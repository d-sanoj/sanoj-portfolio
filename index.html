FinAgent

A conversational finance assistant that turns everyday banking activity into clear, private, plain-language insight.

Objective

The objective of FinAgent is to create a private AI-powered personal finance assistant that lets users ask questions about bank accounts, spending, balances, income, and transactions in plain English through Telegram.

It connects to bank data using SimpleFin and uses an LLM backend such as Ollama for local/private inference or Groq for faster cloud-based responses. The main goal is to make personal finance tracking easier, faster, and more conversational without needing to open multiple banking apps.

Process

The project was built around a two-phase AI workflow. First, the user asks a natural-language financial question, such as "How much did I spend on Amazon this year?" The LLM does not directly calculate the answer; instead, it extracts the user's intent into structured JSON, including the type of query, merchant or category, and date range.

Then deterministic Python code uses Pandas to run the actual calculations on the transaction data. This makes the system more reliable because the LLM handles language understanding, while tested Python logic handles financial computation.

Financial data is pulled from SimpleFin, saved locally as Parquet files, and loaded into the chatbot engine for analysis. The project includes separate modules for syncing data, loading transactions, running the chat engine, and handling Telegram/WhatsApp bot interactions.

Tools
Python Telegram Bot API SimpleFin API Ollama Groq Pandas PyArrow Parquet Flask Requests python-telegram-bot python-dotenv Raspberry Pi Databricks Azure data lake pyspark Antigravity
Value Proposition

FinAgent creates value by turning personal finance analysis into a simple chat experience. Instead of manually checking multiple bank apps or spreadsheets, users can ask questions like "What are my top spending categories this year?" or "How much did I spend on Netflix in the last 3 months?" and receive quick answers based on real transaction data.

It also improves privacy and control. With Ollama, financial data and questions can run fully locally, meaning sensitive information does not need to leave the user's machine. Even when using Groq, transaction data remains local and only the question text is sent for processing. The bot is also owner-locked so only an approved Telegram user ID can access it.

Overall, the project provides a practical, privacy-focused AI assistant for personal finance management. It saves time, reduces the effort needed to analyze spending, gives faster access to financial insights, and demonstrates how LLMs can be combined with deterministic data processing to create reliable real-world applications.

Web / History

Turing > Today

A guided AI-history experience that makes technical milestones easier to scan, understand, and connect to modern AI systems.

Objective

The objective of Turing > Today is to present the complete history of artificial intelligence as a clean, visual, scrollable timeline. It explains AI as a long evolution, starting from Alan Turing, early computation, neural ideas, and the Dartmouth Conference, then moving through symbolic AI, expert systems, AI Winters, machine learning, deep learning, transformers, generative AI, frontier models, AI agents, and AI regulation.

Process of Creating It

The project was created by researching key milestones in AI history and organizing them into a chronological timeline. The website was then built using HTML, CSS, and JavaScript, where data.js stores the timeline content and script.js dynamically displays the cards, people, companies, references, and popups. CSS was used to make the design clean, responsive, and easy to navigate on both desktop and mobile.

Tools Used
HTML CSS JavaScript GitHub GitHub Pages .nojekyll browser DOM APIs CSS media queries OpenAI Codex
Value It Creates

Turing > Today creates value by making AI history easier to understand through a visual, chronological experience instead of a long plain-text article. It helps users see how AI developed step by step, from early theory and symbolic systems to modern foundation models, GPUs, cloud infrastructure, multimodal systems, reasoning models, AI agents, and regulation.

It is also useful as an educational portfolio project because it combines historical research, source-backed content, interactive web design, responsive layout, and storytelling. The project highlights major researchers, builders, labs, and companies such as Alan Turing, John McCarthy, Geoffrey Hinton, Yann LeCun, Fei-Fei Li, OpenAI, Anthropic, Google DeepMind, NVIDIA, Microsoft, Meta, IBM, Hugging Face, DeepSeek, and the European Union.
